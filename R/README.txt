# Praticando R

Bem-vindo ao meu repositório de estudos em R!

Este repositório foi criado para registrar minha evolução durante o aprendizado da linguagem R. Aqui reúno exercícios, exemplos e práticas desenvolvidos à medida que estudo novos conceitos.

## Objetivo

O objetivo deste repositório é consolidar meu aprendizado por meio da prática, explorando desde os conceitos básicos até conteúdos mais avançados da linguagem.

## Organização

Os arquivos estão organizados por temas para facilitar o estudo e acompanhar minha evolução.

- Criação de Matrizes e Data Frames
- Manipulação de Dados
- Vetores *(em breve)*
- Listas *(em breve)*
- Estruturas Condicionais *(em breve)*
- Laços de Repetição *(em breve)*
- Funções *(em breve)*
- Outros tópicos conforme meu aprendizado evoluir.

## Observações

Os exemplos presentes neste repositório utilizam nomes, números e dados **fictícios**, criados exclusivamente para fins de estudo e prática da linguagem R.

Este repositório será atualizado continuamente conforme eu aprender novos conceitos e desenvolver novos exercícios.
