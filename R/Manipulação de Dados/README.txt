# Manipulação de Dados

Esta seção reúne exercícios e exemplos voltados à manipulação de matrizes e data frames na linguagem R.

## Conteúdos praticados

- Acesso a linhas e colunas
- Seleção de células específicas
- Utilização de índices e nomes para acessar dados
- Soma de linhas com `rowSums()`
- Soma de colunas com `colSums()`
- Média de linhas com `rowMeans()`
- Média de colunas com `colMeans()`
- Soma de vetores com `sum()`
- Média de vetores com `mean()`
- Manipulação de dados em matrizes e data frames

O objetivo desta pasta é praticar os principais comandos utilizados para acessar, consultar e realizar operações básicas sobre estruturas de dados em R, consolidando os conceitos fundamentais da linguagem.
