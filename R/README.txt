# Estudos em R

Bem-vindo ao meu repositório de estudos em **R**!

Este repositório foi criado com o objetivo de armazenar minhas práticas, exercícios e projetos desenvolvidos durante o aprendizado da linguagem R. A ideia é registrar minha evolução de forma organizada, documentando os conceitos que estudo e pratico conforme avanço nos meus conhecimentos.

Todos os exercícios utilizam dados fictícios criados exclusivamente para fins de estudo e têm como objetivo fortalecer minha lógica de programação e minha familiaridade com a linguagem.

## Objetivos

- Praticar os fundamentos da linguagem R;
- Resolver exercícios para consolidar o aprendizado;
- Organizar os estudos por tópicos;
- Registrar minha evolução ao longo da aprendizagem;
- Criar um histórico dos conteúdos estudados.

## Conteúdos estudados até o momento

- Criação de matrizes (`matrix`);
- Criação de data frames (`data.frame`);
- Manipulação de dados;
  - Acesso a linhas e colunas;
  - Seleção de elementos;
  - Soma de linhas e colunas (`rowSums()` e `colSums()`);
  - Média de linhas e colunas (`rowMeans()` e `colMeans()`);
  - Cálculo de médias com `mean()`;
- Operadores aritméticos;
  - Multiplicação (`*`);
- Estruturas condicionais;
  - `if`;
  - `if...else`;
  - `if...else if...else`.

## Organização do repositório

Cada pasta representa um tópico estudado e contém:

- `README.md` com uma breve descrição do conteúdo;
- `exercicios.qmd` com os enunciados dos exercícios;
- `resolucao_exercicios.R` com as resoluções em R;
- Pasta `images/` para armazenar tabelas e imagens utilizadas nos exercícios.

## Observações

Este repositório será atualizado continuamente conforme eu aprender novos conceitos e desenvolver novos exercícios na linguagem R, permitindo acompanhar minha evolução de forma organizada e progressiva.
